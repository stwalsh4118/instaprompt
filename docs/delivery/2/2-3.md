## Requirements

1. Add keybindings contribution to package.json
2. Default keybinding: `Ctrl+Shift+I` (Windows/Linux), `Cmd+Shift+I` (Mac)
3. Bind to `instaprompt.selectPrompt` command
4. Keybinding should be customizable via VS Code settings

## Implementation Plan

1. Add `contributes.keybindings` section to package.json
2. Define keybinding:
   ```json
   {
     "command": "instaprompt.selectPrompt",
     "key": "ctrl+shift+i",
     "mac": "cmd+shift+i"
   }
   ```
